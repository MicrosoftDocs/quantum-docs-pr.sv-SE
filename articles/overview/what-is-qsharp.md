## <a name="how-does-q-work"></a><span data-ttu-id="1b32c-125">Hur fungerar Q#?</span><span class="sxs-lookup"><span data-stu-id="1b32c-125">How does Q# work?</span></span>

<span data-ttu-id="1b32c-126">Med programmeringsspråket Q# får du en intuitiv uppsättning typer, åtgärder och logiska uttryck när du ska utveckla algoritmer, utan att du behöver oroa dig för den interna logiken i kvantdatorn.</span><span class="sxs-lookup"><span data-stu-id="1b32c-126">The Q# programming language provides you with an intuitive set of types, operations, and logic expressions to develop algorithms without having to worry about the internal logic of the quantum computer.</span></span>
